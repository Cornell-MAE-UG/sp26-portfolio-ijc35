The most useful Heat Transfer assignment has been Problem Set #5. In problem one of this problem set we used the fin analysis workflow to first solve for temperature along a single fin, and then the single fin effectiveness and efficiency. Once those values were known we then solved for the efficiency and thermal resistance of a fin array. Finally, in problem one we created a thermal resistance network to model a scenario which used the fin array to dissipate heat. We proved that with a fin array, much more heat is dissipated compared to without one. This entire problem combined multiple concepts we have been working on in class including fin analysis and thermal resistance networks. In problem two we applied the lumped capacitance method to solve for the temperature as a function of time of a wire dipped in a bucket of oil. This problem also involved using the Biot number and time constant tau to solve for the wire temperature.
